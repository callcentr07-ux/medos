# МедОсь — клиника биохакинга (лендинг)

Одностраничный сайт клиники биохакинга и превентивной медицины «МедОсь» (г. Сергиев Посад).

Светлый премиум-дизайн, анимированный фон (молекулярная сеть + параллакс медицинских иллюстраций), адаптивная вёрстка. Полностью самодостаточный: один HTML-файл + одно изображение, без сборки и зависимостей.

## Состав

- `index.html` — весь сайт (HTML + CSS + JS в одном файле)
- `kapelnica10-scaled.webp` — изображение капельницы для первого экрана

## Как открыть локально

Просто откройте `index.html` в браузере.

## Публикация (GitHub Pages)

1. Загрузите файлы в репозиторий (см. инструкцию ниже).
2. Settings → Pages → Source: `Deploy from a branch` → ветка `main`, папка `/ (root)` → Save.
3. Через 1–2 минуты сайт будет доступен по адресу `https://<ваш-логин>.github.io/<имя-репозитория>/`.

## Технологии

Чистый HTML5, CSS3 (анимации, backdrop-filter), ванильный JavaScript (Canvas API, IntersectionObserver). Шрифты — Manrope и Unbounded (Google Fonts).
